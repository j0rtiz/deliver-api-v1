# deliver-api-v1

Teste prático Deliver API versão 1

## Install the dependencies

```bash
npm install
```

### Start the api in development mode

```bash
npm run dev
```

### Lint the files

```bash
npm run lint
```

### Start the api in production mode

```bash
npm run start
```
